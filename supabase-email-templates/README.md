# Emerald Coast Marketing Wave - Email Templates

These custom email templates are designed for use with Supabase Authentication.

## Installation Instructions

1. Go to your Supabase Dashboard
2. Navigate to **Authentication → Email Templates**
3. For each template type (Confirmation, Password Recovery, Magic Link), click **Edit**
4. Copy the corresponding HTML template from this folder
5. Paste it into the Supabase template editor
6. Click **Save**

## Template Files

- `confirmation-email.html` - Welcome email with email verification link
- `password-recovery.html` - Password reset email
- `magic-link.html` - Magic link sign-in email

## Brand Colors Used

These templates use your brand's emerald green color scheme:

- Secondary: `#15a893` (lighter emerald)
